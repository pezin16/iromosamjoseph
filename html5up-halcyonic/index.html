My Resume

Tableau

Tableau is a leading Business Intelligence(BI) and data visualization tool, it empowers individuals and organizations to transform raw data into interactive dashboards, providing insights that drive informed decision-making.

Excel

Excel is a versatile spreadsheet software developed by Microsoft, a powerful data visualization and analysis tool used globally. its key features include data entry, computation capabilities, graphing tools, pivot tables.

Power BI

Power BI is a business intelligence and analytics tool developed by microsoft that allows you to connect, analyze, and visualize data from multiple sources. Also it hepls in reporting the transformation of complex datasets into useful informations.

MYSQL

MYSQL IS A relational database organizes data into one or more data tables in which data may be related to each other; these relations help structure the data. SQL is a language that programmers use to create, modify and extract data from the relational database, as well as control user access to the database.

R

R is a programming language for statistical computing and data visualization. It has been adopted in the fields of data mining, bioinformatics and data analysis.

Python

Python is a programming language that can be used for data analysis due to its in-built mathematical libraries and functions like Matplotlibs,Seaborn,Pandas and NumPy to Load,Clean,Prepare and Visualize Datas.
